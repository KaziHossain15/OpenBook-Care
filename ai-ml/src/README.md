# AI/ML Source

Use this folder for draft implementation work.

Current draft structure follows the design note and keeps responsibilities separated.

Planned areas:
- `contracts/` for request/response DTOs and error codes
- `domain/` for core chat/disclaimer entities
- `application/` for controller and explanation coordination logic
- `infrastructure/` for storage, prompt building, and provider gateways

This scaffold is intentionally lightweight and may be reorganized when the team merges work.
